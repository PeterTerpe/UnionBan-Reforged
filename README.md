若无额外偏好，推荐使用Go语言进行编写。因为其容易上手，编译和并行处理更方便。

- Peer相关：
  - 发现相邻节点
  - DHT (libp2p)
  - 证书与签名，信任等级
  - 查询其他Peer的数据库
- API：供网页面板和其他Peer使用
