# Contributing Code

* If you are a Collaborator, please develop on the `dev` branch or create a separate branch, then submit a PR to `main` after testing.
* If you are not a Collaborator, you can contact me to join, or develop directly on your own fork and submit a PR to `main`.
* When possible, please use English comments in the code, and update the corresponding development documentation or README.
* If you would like to improve the documentation, you are also welcome to submit a PR.
* Please try to follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) for commit messages, or refer to the commit style used in this repository’s history.
