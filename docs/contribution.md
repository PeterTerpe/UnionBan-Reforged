# 贡献代码
- 如果你是Collaborator，请在dev分支或另外新建分支开发，测试后发送PR到Main。
- 如果你不是Collaborator，可以联系我来加入，或直接在你的Fork上开发并发送PR到Main。
- 如果可能，代码内尽量使用英文注释，并更新对应的开发文档或README。
- 如果你像对文档补充，也可以发送PR。
- Commit消息请尽量遵循[Conventional Commit](https://www.conventionalcommits.org/en/v1.0.0/)或参考此仓库历史Commit习惯。
